# TutoosNode

Repositorio de prácticas y tutoriales con **Node.js**, enfocado en conceptos básicos e intermedios usando **JavaScript**.

La idea de este proyecto es ir construyendo ejemplos por **clases / sesiones**, para aprender y reforzar conceptos de backend con Node.

---
 
## Requisitos

Antes de comenzar asegúrate de tener instalado:

- **Node.js** (recomendado: versión LTS)
- **npm** (se incluye con Node.js)
- Git (opcional, pero recomendado)

---

## Instalación

Clona el repositorio:

```bash
git clone https://github.com/omarbermejo/TutoosNode.git
cd TutoosNode
```

Instala dependencias (si tu clase/proyecto lo incluye):

```bash
npm install
```

> Nota: Algunas carpetas pueden ser ejemplos simples sin dependencias.

---

## Ejecución

Dependiendo del ejercicio/clase:

### Ejecutar un archivo individual

```bash
node nombreDelArchivo.js
```

Ejemplo:

```bash
node index.js
```

### Ejecutar una carpeta/clase específica

Si estás trabajando dentro de una carpeta, como por ejemplo:

```bash
cd "class 1"
node index.js
```

- Mantén cada clase como una carpeta independiente.
- Documenta cada ejercicio con un comentario o mini README dentro de cada carpeta.
